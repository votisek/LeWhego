# Hex code indexes:
- ## 0x1 == Volant
    - Counts from 0
    > ### All values
    > ___
    > Values are indexed from 0
    > -90° = 0x100
    > 0° = 0x5a
    > 90° = 0xb4
___
- ## 0x2 == Padla
    - 4 commands with off
    > ### All values
    > ___
    > 0x200 = [0, 0, 0]
    > 0x211 = [1, 0, 0] = Slowed time
    > 0x221 = [0, 1, 0] = Jump
    > 0x231 = [0, 0, 1] = Both at the same time
